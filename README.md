# Nishalan Govender

**Robotics software engineer. I solve problems — starting at the whiteboard, not the keyboard.**

Mechatronics graduate from Stellenbosch University, working across robotics, embedded systems, and full-stack web. I walk the floor, understand the process, then design modular systems that scale — deploying a proof of concept as fast as possible, then iterating. The tech stack changes; the approach doesn't.

Currently freelancing and open to full-time roles — remote, on-site, or relocation.

## Selected work

- **[nishalangovender.com](https://nishalangovender.com)** — engineering portfolio with interactive project demos. Next.js 16, React 19, Tailwind v4, MDX blog, Framer Motion.
- **NishOS** — personal life-management system on Supabase + Next.js. Replaces a Notion-based LifeOS with a typed schema, full-text search, and a knowledge-graph wiki layer. Repo is private; the architecture is documented at [nishalangovender.com](https://nishalangovender.com).
- **[park_bot](https://github.com/nishalangovender/park_bot)** — autonomous parking of a 4WS vehicle in ROS2 + Ignition Gazebo. My Mechatronics Project 488 thesis (Stellenbosch, Cum Laude 2022), recently re-polished to a senior-engineer standard: CI, unit tests against a documented kinematic reference, headless Docker, and a live TypeScript demo on the site.
